## 2. UI 专业度问题汇总

### 2.1 视觉层级问题

| 问题 | 描述 | 建议 |
|-----|------|------|
| 字体层级不清晰 | 标题与正文字号差异小 | 标题18px→20px，正文14px→13px |
| 颜色对比度不足 | 部分灰色文字偏淡 | 使用#666代替#999 |
| 间距不统一 | 不同页面间距差异大 | 统一使用4/8/12/16/24px间距体系 |

### 2.2 交互反馈问题

| 问题 | 描述 | 建议 |
|-----|------|------|
| 按钮点击反馈弱 | 缺少按压态 | 添加:active状态，透明度变化或缩放 |
| 列表项点击区域小 | 可点击区域不明确 | 增大点击区域至整个卡片 |
| 加载状态缺失 | 图片加载中无反馈 | 添加骨架屏或loading动画 |
| 操作成功反馈 | 收藏成功提示不明显 | 使用toast+微动效 |

### 2.3 图片处理问题

| 问题 | 描述 | 建议 |
|-----|------|------|
| 图片缺失占位 | 显示空白或灰色背景 | 使用渐变色+图标+文字占位 |
| 图片比例不一 | 不同图片显示效果差异 | 统一使用object-fit: cover |
| 图片加载失败 | 无错误处理 | 添加onError事件，显示默认图 |

### 2.4 布局适配问题

| 问题 | 描述 | 建议 |
|-----|------|------|
| 安全区适配 | 底部tab栏未适配刘海屏 | 添加env(safe-area-inset-bottom) |
| 横屏适配 | 未考虑横屏情况 | 添加横屏检测，提示竖屏使用 |
| 字体缩放 | 系统字体放大后布局错乱 | 使用rpx单位，避免固定高度 |

## 4. 设计规范建议

### 4.1 颜色规范

```scss
// 主色调
$brand-primary: #4CAF50;      // 品牌绿
$brand-dark: #388E3C;         // 深绿（按压态）
$brand-light: #C8E6C9;        // 浅绿（背景）

// 文字色
$text-primary: #333333;       // 主文字
$text-secondary: #666666;     // 次要文字
$text-tertiary: #999999;      // 辅助文字

// 背景色
$bg-page: #F5F5F5;            // 页面背景
$bg-card: #FFFFFF;            // 卡片背景
$bg-hover: #F0F0F0;           // 悬停背景

// 功能色
$accent-orange: #FF9800;      // 评分/收藏
$warning: #FFC107;            // 警告/提示
$error: #F44336;              // 错误
```

### 4.2 字体规范

| 层级 | 字号 | 字重 | 用途 |
|-----|------|------|------|
| 大标题 | 20px | Bold | 页面标题 |
| 标题 | 18px | Bold | 区块标题 |
| 小标题 | 16px | Medium | 卡片标题 |
| 正文 | 14px | Regular | 正文内容 |
| 辅助 | 12px | Regular | 辅助信息 |
| 标注 | 10px | Regular | 时间/标签 |

### 4.3 间距规范

| 名称 | 数值 | 用途 |
|-----|------|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 紧凑间距 |
| md | 12px | 默认间距 |
| lg | 16px | 区块间距 |
| xl | 24px | 大区块间距 |
| xxl | 32px | 页面间距 |

### 4.4 圆角规范

| 名称 | 数值 | 用途 |
|-----|------|------|
| sm | 4px | 小按钮、标签 |
| md | 8px | 卡片、输入框 |
| lg | 12px | 大卡片、弹窗 |
| full | 50% | 头像、圆形按钮 |
