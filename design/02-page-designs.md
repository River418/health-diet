# Health-Diet 页面设计说明

## 1. 首页/发现页

### 1.1 现有问题分析
- 搜索栏过于简单，缺乏视觉层次
- 分类图标尺寸小、辨识度低
- 页面空白区域过多，内容密度低
- 养生资讯区域缺乏吸引力
- 底部导航选中状态不明显

### 1.2 重新设计方案

#### 布局结构

│  状态栏                          │

#### 视觉优化点
1. **顶部品牌区**：增加品牌Logo和slogan，提升专业感
2. **搜索栏优化**：
   - 圆角增大至24px
   - 添加搜索图标
   - 占位符文字增加引导性
   - 背景使用浅灰色(#F5F5F5)
3. **轮播Banner**：
   - 高度160px
   - 圆角12px
   - 渐变遮罩保证文字可读性
   - 自动轮播 + 手动滑动
4. **分类入口**：
   - 图标尺寸增大至48px
   - 使用渐变色背景圆形图标
   - 文字字号14px，深灰色
   - 增加点击反馈动画
5. **推荐配方卡片**：
   - 横向滚动展示
   - 卡片宽度160px
   - 图片高度120px，圆角12px
   - 标题16px，功效标签12px
   - 评分使用橙色星星
6. **养生资讯**：
   - 左图右文布局
   - 图片80×80px圆角8px
   - 标题16px，摘要14px灰色
   - 添加阅读量和阅读时间

#### 交互优化
- 下拉刷新：显示品牌Logo动画
- 上拉加载：底部loading动画
- 卡片点击：轻微缩放反馈
- 分类入口：点击后图标放大+颜色变化

3. **标题区**：
   - 标题22px Bold

5. **制作步骤**：
   - 步骤数字使用品牌绿圆形背景
   - 文字左对齐，步骤图可点击放大
   - 步骤间用细线分隔
6. **用户评论**：
   - 展示3条精选评论
   - 头像+昵称+内容+评分+时间
   - 点击跳转评论列表
7. **底部操作栏**：
   - 固定在底部
   - 收藏、评论按钮在左
   - 「写评论」主按钮在右

#### 交互优化
- 图片点击：全屏查看，支持缩放
- 步骤图：点击放大，左右滑动切换
- 收藏按钮：点击有爱心动画
- 食材项：左滑显示「加入购物清单」

## 6. 通用组件优化

### 6.1 底部导航栏
```
┌─────────────────────────────────┐
│  🏠        📂        👤        │
│ 首页      分类      我的        │
└─────────────────────────────────┘
```
- 高度：56px + 安全区域
- 背景：白色 + 上边框阴影
- 选中：品牌绿图标+文字
- 未选中：灰色图标+文字
- 图标尺寸：24px
- 文字：12px

### 6.2 提示条（Disclaimer）
```
┌─────────────────────────────────┐
│ ⚠️ 食疗养生，不能代替药物治疗...  │
└─────────────────────────────────┘
```
- 背景：浅黄色 #FFF8E1
- 边框：1px solid #FFE082
- 文字：#F57C00
- 字号：12px
- 圆角：8px
- 内边距：12px 16px

### 6.3 加载状态
- 骨架屏：灰色渐变动画
- Loading图标：品牌绿旋转动画
- 空状态：插画+文字引导

### 6.4 错误状态
- 错误图标：灰色感叹号
- 错误文字：14px灰色
- 重试按钮：主按钮样式
