# 首页设计稿

## 1. 页面概述

首页是 Health-Diet 小程序的主入口，包含金刚位快捷筛选、搜索栏、轮播Banner、推荐配方和信息流内容。

---

## 4. 页面状态

### 4.1 加载状态

```
┌─────────────────────────────────────┐
│                                     │
│  ┌─────────────────────────────┐   │
│  │  █████████████████████████  │   │  ← 骨架屏
│  └─────────────────────────────┘   │
│                                     │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐      │
│  │████│ │████│ │████│ │████│      │
│  └────┘ └────┘ └────┘ └────┘      │
│  ████████ ████████                │
│                                     │
└─────────────────────────────────────┘
```

- 显示骨架屏占位
- 金刚位显示灰色占位块
- 卡片区域显示卡片骨架

### 4.2 空状态

- 信息流为空时显示空状态插画
- 提示"暂无推荐内容"
- 提供"刷新"按钮

### 4.3 错误状态

- 网络错误时显示错误提示
- 提供"重新加载"按钮
- Toast提示"网络连接失败"

### 4.4 下拉刷新

- 下拉距离: 60px
- 刷新图标: 旋转动画
- 文案: "下拉刷新" → "释放刷新" → "刷新中..."
