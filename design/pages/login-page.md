# 登录页设计稿

## 1. 页面概述

登录页提供微信一键登录和抖音一键登录功能，是用户进入小程序的入口页面。设计简洁友好，降低用户登录门槛。

## 4. 登录流程状态

### 4.1 初始状态

- 显示 Logo、登录按钮、协议文字
- 按钮可点击
- 无加载动画

### 4.2 加载状态

```
┌─────────────────────────────────────┐
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │    ○  登录中...             │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

- 按钮显示加载动画
- 按钮文字变为"登录中..."
- 按钮禁用，透明度 70%
- 其他按钮禁用

### 4.3 成功状态

- Toast提示"登录成功"
- 自动跳转回上一页或首页
- 转场动画: 淡入 200ms

### 4.4 失败状态

- Toast提示错误信息
- 按钮恢复可点击状态
- 错误提示:
  - 网络错误: "网络连接失败，请重试"
  - 用户取消: 无提示
  - 服务错误: "登录服务繁忙，请稍后再试"
