# Health-Diet 图片规范

## 图片概述

Health-Diet 使用统一的图片规范，确保视觉一致性和加载性能。

## 图片类型

| 类型 | 用途 | 格式 |
|------|------|------|
| 配方封面 | 配方列表展示 | JPG/WebP |
| 配方详情 | 配方详情页大图 | JPG/WebP |
| 步骤图片 | 制作步骤说明 | JPG/WebP |
| 食材图片 | 食材展示 | PNG/WebP |
| 用户头像 | 用户头像 | JPG/PNG |
| Banner | 运营位展示 | JPG/WebP |
| 内容封面 | 信息流内容 | JPG/WebP |
| 图标 | UI图标 | SVG/PNG |

## 图片压缩规范

### JPG 压缩
- 质量: 80-85%
- 渐进式: 是
- 色彩空间: sRGB

### WebP 压缩
- 质量: 80%
- 有损压缩
- 支持透明度

### PNG 压缩
- 使用 TinyPNG 等工具压缩
- 保留透明度
- 8-bit 颜色深度 (如可能)

## 图片命名规范

```
[type]_[id]_[size].[ext]

示例:
recipe_123_cover.jpg
recipe_123_step_1.jpg
ingredient_red dates_thumb.png
user_456_avatar.jpg
banner_home_1.jpg
```

## 图片存储规范

### 目录结构

```
images/
├── recipes/
│   ├── covers/           # 配方封面
│   ├── steps/            # 步骤图片
│   └── thumbs/           # 缩略图
├── ingredients/          # 食材图片
├── users/                # 用户头像
├── banners/              # Banner图
├── contents/             # 内容封面
└── placeholders/         # 占位图
```

### CDN 路径

```
https://cdn.health-diet.com/images/[type]/[filename]

示例:
https://cdn.health-diet.com/images/recipes/covers/recipe_123_cover.jpg
```

## 图片加载策略

### 懒加载
- 首屏外图片使用懒加载
- 占位图: 模糊渐进式加载

### 响应式图片
- 使用 srcset 提供多尺寸
- 根据设备像素比选择

```html
<img src="image_1x.jpg"
     srcset="image_1x.jpg 1x, image_2x.jpg 2x, image_3x.jpg 3x"
     alt="描述文字">
```

### 渐进式加载
1. 显示模糊占位图 (低质量预览)
2. 加载中显示骨架屏
3. 加载完成显示高清图
4. 淡入动画过渡

## 图片占位图

### 配方占位图
- 背景色: #F5F5F5
- 图标: 图片图标 (48px, #CCCCCC)
- 文字: "图片加载中" (14px, #999999)

### 头像占位图
- 背景色: #E8F5E9
- 默认头像图标 (用户图标)
- 颜色: #4CAF50

### 空状态占位图
- 插画风格
- 尺寸: 120px x 120px
- 颜色: #CCCCCC

|------|--------|

## 图片适配规范

### 响应式适配
- 使用 object-fit: cover
- 保持图片比例
- 裁剪多余部分

### 深色模式适配
- 图片亮度调整
- 使用 CSS filter: brightness(0.9)

## 图片版权规范

- 所有图片必须有版权授权
- 用户上传图片需审核
- 违规图片及时处理
- 保留图片来源信息

## 图片性能优化

### 文件大小
- 配方封面: < 200KB
- 步骤图片: < 150KB
- 食材图片: < 50KB
- 用户头像: < 30KB
- Banner: < 300KB

### 加载优化
- 使用 CDN 加速
- 开启 HTTP/2
- 图片预加载关键资源
- 使用 Service Worker 缓存
