# Health-Diet 组件库规范

## 1. 组件总览

本文档定义 Health-Diet 小程序的组件设计规范，包括组件结构、样式、交互和状态。

---

## 2. 基础组件 (Basic Components)

### 2.1 Button 按钮

#### 按钮类型

| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 填充主色背景，白色文字 | 主要操作、提交、确认 |
| Secondary | 填充辅助色背景，白色文字 | 次要操作、强调 |
| Outline | 透明背景，主色边框和文字 | 辅助操作、取消 |
| Ghost | 透明背景，主色文字 | 弱操作、链接样式 |
| Text | 纯文字，主色 | 文字链接、极简操作 |
| Danger | 填充红色背景，白色文字 | 删除、危险操作 |

#### 按钮尺寸

| 尺寸 | 高度 | 内边距 | 字号 | 用途 |
|------|------|--------|------|------|
| Small | 32px | 8px 16px | 13px | 紧凑空间、内联按钮 |
| Medium | 44px | 12px 24px | 15px | 标准按钮 |
| Large | 52px | 16px 32px | 16px | 重要操作、底部固定 |
| XLarge | 56px | 16px 40px | 17px | 大字模式、适老化 |

#### 按钮状态

| 状态 | 样式变化 |
|------|---------|
| Default | 标准样式 |
| Hover | 亮度提升 5% |
| Active | 亮度降低 10% |
| Disabled | 透明度 50%，cursor: not-allowed |
| Loading | 显示加载动画，禁用点击 |

#### 按钮组合

```
┌─────────────────────────────────────┐
│  [取消]        [确认]              │  ← Outline + Primary
├─────────────────────────────────────┤
│  [    主要操作按钮    ]              │  ← 通栏按钮
├─────────────────────────────────────┤
│  [图标] 带图标按钮                   │  ← Icon + Text
└─────────────────────────────────────┘
```

## 5. 组件状态规范

### 5.1 通用状态

| 状态 | 说明 |
|------|------|
| Default | 默认状态 |
| Hover | 悬停状态 (桌面端) |
| Active | 按下/激活状态 |
| Focus | 聚焦状态 |
| Disabled | 禁用状态 |
| Loading | 加载状态 |
| Error | 错误状态 |
| Success | 成功状态 |

### 5.2 状态样式规则

```css
/* Default */
opacity: 1;
transform: scale(1);

/* Hover */
filter: brightness(1.05);

/* Active */
filter: brightness(0.95);
transform: scale(0.98);

/* Disabled */
opacity: 0.5;
pointer-events: none;

/* Loading */
opacity: 0.7;
pointer-events: none;
```

## 6. 组件命名规范

### 6.1 BEM 命名法

```
Block__Element--Modifier

示例:
.btn                    // 按钮块
.btn__icon             // 按钮图标元素
.btn--primary          // 主要按钮修饰符
.btn--large            // 大尺寸修饰符
.btn--disabled         // 禁用状态修饰符
```

### 6.2 组件前缀

| 前缀 | 说明 |
|------|------|
| hd- | Health-Diet 通用组件 |
| hd-btn | 按钮组件 |
| hd-card | 卡片组件 |
| hd-list | 列表组件 |
| hd-modal | 弹窗组件 |
| hd-form | 表单组件 |
| hd-nav | 导航组件 |
