## 1. 产品概述

### 1.1 产品定位

Health-Diet 是一款**面向中老年和白领群体的养生食疗配方查询小程序**，提供科学、实用的食疗养生方案，帮助用户通过日常饮食调理身体、预防疾病。产品支持微信小程序、抖音小程序双平台，并可打包为独立 iOS/Android App。

**核心定位关键词**:
- **科学养生**: 基于中医理论和现代营养学的食疗方案
- **便捷查询**: 多维度分类筛选，快速找到适合的食疗配方
- **双平台覆盖**: 微信小程序 + 抖音小程序，触达不同用户群体
- **内容聚合**: 对接微信视频号/公众号、抖音内容，提供丰富的养生资讯

### 1.2 目标用户

#### 1.2.1 主要用户群体

| 用户类型 | 特征 | 核心需求 | 价值评估 |
|---------|------|---------|---------|
| **中老年用户** | 50-70岁，关注健康养生，有慢性病调理需求 | 简单易懂的操作界面，针对常见老年病的食疗方案 | 高活跃度，强分享意愿 |
| **白领上班族** | 25-40岁，工作压力大，亚健康状态普遍 | 快速缓解疲劳、改善睡眠、调节肠胃的食疗方案 | 中等付费意愿，注重效率 |

#### 1.2.2 次要用户群体

- **养生爱好者**: 主动学习养生知识，关注节气养生
- **家庭主妇/主夫**: 为家人准备健康餐食
- **慢性病患者**: 寻求辅助调理方案（需明确标注不替代药物治疗）

### 1.3 核心价值主张

#### 1.3.1 统一养生生态

- 搜索结果页底部提示

```

                                                      │
                                                      ▼
                                              [生成 JWT Token]
                                                      │
                                                      ▼
                                              [返回登录成功]
```

**验收标准**:

## 3. 核心功能详细需求

### 3.1 用户系统

#### 3.1.1 微信登录

**用户故事**:
> 作为微信小程序用户，我希望一键登录，无需输入账号密码。

**功能流程**:
```
[点击微信登录] → [调用 wx.login] → [获取 code] → [后端换取 openid/session_key]
      │                                                          │
      ▼                                                          ▼
  显示微信登录按钮                                        [创建用户记录]
                                                                 │
                                                                 ▼
                                                          [生成 JWT Token]
                                                                 │
                                                                 ▼
                                                          [返回登录成功]
```

**用户数据结构**:
```typescript
interface User {
  id: string;
  wxOpenId?: string;         // 微信 openid
  dyOpenId?: string;         // 抖音 openid
  nickname: string;          // 昵称
  avatar: string;            // 头像URL
  phone?: string;            // 手机号
  language: 'zh-CN' | 'zh-TW' | 'en';  // 偏好语言
  createdAt: Date;
  updatedAt: Date;
}
```

**验收标准**:
- [ ] 首次登录自动创建用户
- [ ] 重复登录返回已有用户
- [ ] 登录状态保持7天

**优先级**: P0

**用户故事**:
> 作为抖音小程序用户，我希望一键登录，使用抖音账号授权。

**功能流程**:
```
[点击抖音登录] → [调用 tt.login] → [获取 code] → [后端换取 openid]
```

**验收标准**:
- [ ] 支持抖音小程序登录

- [ ] 登录流程与微信保持一致

**优先级**: P0

#### 3.1.3 语言切换

```typescript
interface Recipe {
```
