## 1. 产品概述

### 1.1 产品定位

Health-Diet 是一款**面向中老年和白领群体的养生食疗配方查询小程序**，提供科学、实用的食疗养生方案，帮助用户通过日常饮食调理身体、预防疾病。产品支持微信小程序、抖音小程序双平台，并可打包为独立 iOS/Android App。

**核心定位关键词**:
- **科学养生**: 基于中医理论和现代营养学的食疗方案
- **便捷查询**: 多维度分类筛选，快速找到适合的食疗配方
- **双平台覆盖**: 微信小程序 + 抖音小程序，触达不同用户群体
- **内容聚合**: 对接微信视频号/公众号、抖音内容，提供丰富的养生资讯

### 1.2 目标用户

#### 1.2.1 主要用户群体

| 用户类型 | 特征 | 核心需求 | 价值评估 |
|---------|------|---------|---------|
| **中老年用户** | 50-70岁，关注健康养生，有慢性病调理需求 | 简单易懂的操作界面，针对常见老年病的食疗方案 | 高活跃度，强分享意愿 |
| **白领上班族** | 25-40岁，工作压力大，亚健康状态普遍 | 快速缓解疲劳、改善睡眠、调节肠胃的食疗方案 | 中等付费意愿，注重效率 |

#### 1.2.2 次要用户群体

- **养生爱好者**: 主动学习养生知识，关注节气养生
- **家庭主妇/主夫**: 为家人准备健康餐食
- **慢性病患者**: 寻求辅助调理方案（需明确标注不替代药物治疗）

### 2.4 消息队列与异步处理

**功能描述**: 使用消息队列处理异步任务，提升系统响应速度和可靠性。

**技术选型**: RabbitMQ 3.x 或 Kafka 3.x

**应用场景**:

| 场景 | 队列名称 | 处理方式 | 优先级 |
|-----|---------|---------|-------|
| 内容审核 | content.audit | 用户发布评论/图片后异步审核 | 高 |
| 通知推送 | notification | 异步发送系统通知、运营消息 | 中 |
| 数据同步 | data.sync | 跨平台用户数据同步 | 中 |
| 缓存预热 | cache.warmup | 热门数据缓存预热 | 低 |
| 日志处理 | log.process | 异步处理操作日志 | 低 |

**消息处理流程**:
```
[业务操作] → [发送消息到队列] → [消费者处理] → [处理结果]
     │              │                │            │
     ▼              ▼                ▼            ▼
  用户发表评论   写入MQ队列      敏感词过滤      审核通过/拒绝
                              图片鉴黄
                              人工审核
```

**验收标准**:
- [ ] 消息投递成功率 ≥ 99.9%
- [ ] 消息处理延迟 ≤ 5 秒
- [ ] 支持消息重试机制（最多3次）
- [ ] 支持死信队列处理失败消息

### 2.3 后端服务能力

#### 2.3.1 认证服务

**技术方案**: 微信/抖音 OAuth2.0 + JWT Token

**功能点**:

| 功能点 | 描述 | 优先级 |
|-------|------|-------|
| 微信一键登录 | 微信小程序 wx.login 获取 code | P0 |
| 抖音一键登录 | 抖音小程序 tt.login 获取 code | P0 |
| 手机号绑定 | 获取用户手机号（可选） | P1 |
| Token 刷新 | JWT Token 自动刷新机制 | P0 |
| 多端登录 | 支持微信/抖音账号关联 | P2 |

**登录流程**:

### 8.2 异常状态设计规范

#### 8.2.1 加载状态

**骨架屏 (Skeleton)**:
- 使用灰色占位块模拟内容结构
- 背景色：#F5F5F5
- 动画：脉冲动画，透明度0.5-1循环
- 持续时间：最长3秒，超时显示错误状态

**加载动画**:
- 类型：旋转圆圈或脉冲点
- 颜色：主色 #4CAF50
- 尺寸：24px-48px
- 文案："加载中..."

#### 8.2.2 空状态

**通用空状态**:
```
┌─────────────────────────────┐
│                             │
│      [空状态插画]           │
│                             │
│      暂无内容               │
│      这里还没有内容，去      │
│      其他地方看看吧          │
│                             │
│      [操作按钮]             │
│                             │
└─────────────────────────────┘
```

**具体场景**:
| 场景 | 标题 | 描述 | 操作 |
|-----|------|------|------|
| 无搜索结果 | 未找到相关配方 | 换个关键词试试 | 清除搜索 |
| 无收藏 | 暂无收藏 | 看到喜欢的配方就收藏吧 | 去首页 |
| 无评论 | 暂无评论 | 成为第一个评论的人 | 写评论 |
| 无网络 | 网络连接失败 | 请检查网络设置 | 重试 |

- 错误提示：红色文字，位于输入框下方
- 图标：错误图标 + 文字
- 实时校验：失去焦点时校验
