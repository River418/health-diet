# Health-Diet 食疗养生小程序 UI/UX 优化设计方案

> **设计版本**: v2.0 专业版  
> **设计目标**: 提升医疗/健康类产品的专业可信度  
> **目标用户**: 中老年群体 + 白领女性  
> **设计日期**: 2026-03-14

---

## 📋 执行摘要

### 当前UI问题诊断

经过对现有代码的全面审查，发现以下核心问题导致UI显得"不专业"：

| 问题类别 | 具体问题 | 影响 |
|---------|---------|------|
| **色彩系统** | 品牌绿(#2E7D32)饱和度过高，缺乏医疗/健康产品应有的沉稳感 | 显得廉价、不够可信 |
| **视觉层次** | 页面元素间距不统一，信息层级混乱 | 用户难以快速获取关键信息 |
| **字体系统** | 字号跳跃过大(12px→22px)，缺乏中间过渡 | 阅读体验不流畅 |
| **图标使用** | 使用Emoji作为图标(⭐💬❤️)，显得随意 | 降低专业感 |
| **阴影/圆角** | 阴影参数不一致，圆角使用混乱 | 视觉不统一 |
| **图片处理** | 缺乏统一的图片处理规范 | 整体质感参差 |

## 🎨 第一部分：视觉风格升级

### 1.1 新的品牌色彩系统

#### 主色调（医疗级专业绿）

```scss
// 主色 - 医疗级沉稳绿（降低饱和度，提升专业感）
$brand-primary: #3D8B5D;        // 主按钮、导航选中
$brand-primary-hover: #2D7A4D;  // 悬停状态
$brand-primary-pressed: #1E6940; // 按下状态
$brand-primary-light: #E8F5ED;  // 浅色背景

// 辅助色 - 自然疗愈色系
$accent-teal: #5BA8A0;          //  teal绿 - 清新、疗愈
$accent-sage: #8FB996;          // 鼠尾草绿 - 温和、自然
$accent-amber: #D4A574;         // 琥珀棕 - 温暖、传统中医感
$accent-coral: #E07A5F;         // 珊瑚橙 - 活力、健康（慎用）

// 中性色 - 医疗文档级
$text-primary: #1C1C1E;         // 主文字 - 近乎纯黑
$text-secondary: #3A3A3C;       // 次要文字
$text-tertiary: #6B6B6B;        // 辅助文字
$text-placeholder: #9E9E9E;     // 占位文字

// 背景色 - 护眼模式
$bg-page: #F5F7F5;              // 页面背景 - 微绿护眼
$bg-card: #FFFFFF;              // 卡片背景
$bg-surface: #FAFBFA;           // 表面背景
$bg-hover: #F0F4F0;             // 悬停背景

// 功能色 - 医疗标准
$success: #34C759;              // 成功 - Apple绿
$warning: #FF9500;              // 警告 - 橙黄
$error: #FF3B30;                // 错误 - 红
$info: #007AFF;                 // 信息 - 蓝
```

#### 色彩使用规范

| 场景 | 使用颜色 | 说明 |
|-----|---------|------|
| 主按钮 | $brand-primary | CTA按钮、重要操作 |
| 次要按钮 | $brand-primary-light + $brand-primary 文字 | 次要操作 |
| 标签/徽章 | $accent-teal, $accent-sage | 分类标签 |
| 警示信息 | $warning 背景 + $text-primary 文字 | 免责声明 |
| 评分/收藏 | $accent-amber | 星级评分 |

### 1.2 字体系统升级

#### 字体家族

```scss
// 主字体 - 系统字体栈（确保中文显示质量）
$font-family-base: -apple-system, BlinkMacSystemFont, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;

// 数字字体 - 等宽数字，便于阅读数据
$font-family-number: 'SF Pro Display', 'Helvetica Neue', 'PingFang SC', sans-serif;

// 标题字体 - 可选：思源宋体（如需传统中医感）
$font-family-heading: 'Source Han Serif CN', 'Noto Serif SC', serif;
```

#### 字号层级（8级系统）

```scss
// 标题层级
$font-size-hero: 28px;        // 页面大标题（如欢迎页）
$font-size-h1: 24px;          // 页面标题
$font-size-h2: 20px;          // 区块标题
$font-size-h3: 18px;          // 卡片标题
$font-size-h4: 16px;          // 列表标题

// 正文层级
$font-size-body-large: 16px;  // 重要正文（适老化）
$font-size-body: 15px;        // 标准正文
$font-size-body-small: 14px;  // 辅助正文

// 辅助层级
$font-size-caption: 13px;     // 说明文字
$font-size-footnote: 12px;    // 脚注、时间
$font-size-tiny: 11px;        // 标签、徽章
```

#### 字重规范

```scss
$font-weight-regular: 400;    // 正文
$font-weight-medium: 500;     // 按钮、标签
$font-weight-semibold: 600;   // 小标题
$font-weight-bold: 700;       // 大标题
```

#### 行高规范

```scss
$line-height-tight: 1.3;      // 标题
$line-height-normal: 1.5;     // 正文
$line-height-relaxed: 1.7;    // 长文本（适老化）
```

## 📱 第二部分：关键页面优化设计

### 2.1 首页/仪表盘优化

#### 当前问题
- Banner高度160px过小，视觉冲击力不足
- 金刚位使用Emoji图标，不专业
- 推荐配方横向滚动，展示效率低
- 养生资讯卡片信息密度低

#### 优化方案

**布局结构调整：**

```
┌─────────────────────────────────┐
│ 🔍 搜索栏（sticky）               │
├─────────────────────────────────┤
│ ┌─────────────────────────────┐ │
│ │      Banner轮播 (200px)      │ │
│ │    大尺寸美食图片 + 渐变遮罩   │ │
│ └─────────────────────────────┘ │
├─────────────────────────────────┤
│ ┌────┬────┬────┬────┐          │
│ │ 🍵 │ 🌿 │ 👤 │ 📅 │ 金刚位   │
│ │功效│食材│人群│节气│ (图标)   │
│ └────┴────┴────┴────┘          │
├─────────────────────────────────┤
│ 📌 今日推荐                      │
│ ┌────────┬────────┐            │
│ │ 卡片1   │ 卡片2   │ 2列网格   │
│ │ 大图    │ 大图    │           │
│ └────────┴────────┘            │
├─────────────────────────────────┤
│ 📰 养生资讯                      │
│ ┌─────────────────────────────┐ │
│ │ 图文混排卡片（左图右文）      │ │
│ └─────────────────────────────┘ │
├─────────────────────────────────┤
│ ⚠️ 免责声明                      │
└─────────────────────────────────┘
```

**关键改进：**

1. **Banner区域**
   - 高度从160px提升至200px
   - 图片使用高质量美食摄影
   - 渐变遮罩：从底部30%处开始，使用 `linear-gradient(to top, rgba(0,0,0,0.6), transparent)`
   - 标题字号：24px，白色，加粗
   - 副标题：14px，白色80%透明度

2. **金刚位优化**
   - 替换Emoji为专业SVG图标
   - 图标尺寸：48px
   - 背景：渐变圆形（每个分类不同颜色）
   - 文字：13px，$text-secondary

3. **推荐配方**
   - 改为2列网格布局（替代横向滚动）
   - 卡片比例：4:3图片 + 信息区
   - 显示：评分（星级图标）+ 收藏数

4. **养生资讯**
   - 左图右文布局
   - 图片：80x60px，圆角8px
   - 标题：16px，最多2行
   - 元信息：12px，灰色
