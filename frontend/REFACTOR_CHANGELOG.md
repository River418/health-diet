# Health-Diet 前端重构变更说明

## 重构概述

根据 Linda 的设计文档对 health-diet 项目前端进行全面重构，主要目标是提升视觉品质、用户体验和无障碍支持。

**重构日期**: 2026-03-11  
**设计文档版本**: v1.0  
**技术栈**: Taro 3.6.23 + Vue3

---

## 1. 全局样式系统（已完成）

### 变更文件
- `src/styles/variables.scss` - 设计变量系统
- `src/styles/index.scss` - 全局样式

### 主要变更
1. **色彩系统**
   - 主色调：#2E7D32（品牌绿）
   - 辅助色：#FF8C42（暖橙）、#FFD54F（暖黄）、#64B5F6（天蓝）
   - 中性色：#1A1A1A（标题）、#4A4A4A（正文）、#8A8A8A（次要文字）
   - 背景色：#F8F9FA（页面）、#FFFFFF（卡片）

2. **字体系统（中老年友好）**
   - 正文：15px（比常规14px更大）
   - 标题：18-22px
   - 支持「大字模式」

3. **间距系统（基于4px）**
   - xs: 4px, sm: 8px, md: 12px, lg: 16px, xl: 20px, xxl: 24px

4. **圆角系统**
   - 卡片：12px
   - 按钮：24px（全圆角）

5. **阴影系统**
   - 卡片阴影：0 2px 12px rgba(0, 0, 0, 0.06)
   - 悬浮阴影：0 4px 16px rgba(0, 0, 0, 0.1)

## 8. 无障碍支持

1. **字体适配**
   - 正文最小15px

2. **对比度**
   - 文字与背景对比度 ≥ 4.5:1
   - 大文字对比度 ≥ 3:1

3. **触摸目标**
   - 最小点击区域44×44px
   - 按钮高度48px

4. **减少动画**

## 10. 技术实现细节

### 响应式断点
```scss
$breakpoint-sm: 375px;  // 手机小屏
$breakpoint-md: 414px;  // 手机大屏
$breakpoint-lg: 768px;  // 平板
```

### 动画缓动函数
```scss
$ease-standard: ease-in-out;
$ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
$ease-decelerate: cubic-bezier(0, 0, 0.2, 1);
```

### 阴影层级
```scss
$shadow-level-1: 0 1px 3px rgba(0,0,0,0.05);
$shadow-level-2: 0 2px 8px rgba(0,0,0,0.08);
$shadow-level-3: 0 4px 16px rgba(0,0,0,0.12);
$shadow-level-4: 0 8px 24px rgba(0,0,0,0.15);
```
