# Health Diet - 食疗养生小程序

## 项目概述
中老年和白领养生食疗配方查询小程序，支持微信/抖音双平台，可打包为独立 App。

## 核心功能
1. 微信/抖音一键登录
2. 金刚位快速筛选（配置化）
3. 多维度分类（人群/节气/疗效）
4. 信息流（对接视频号/公众号/抖音内容）
5. 三语言切换（简中/繁中/English）
6. 配方详情（图文视频 + 评论 + 合规提示）

## 技术栈
- 跨端框架：UniApp / Taro
- 可打包：iOS App + Android App
